# Onyx - Your AI Agent

Onyx is your personal AI agent that responds to voice commands, grows smarter & more capable over time, and earns you bitcoin.

It's part of the OpenAgents network where every agent makes all agents smarter.

_Heavily work in progress. Beta launch ETA December 20. [Follow along on X.](https://x.com/OpenAgentsInc/status/1864878103853477910)_

![onyx](https://github.com/user-attachments/assets/75f0d9e5-2c66-46fd-8b00-14295e4bb808)

## Features

- Natural voice commands for instant assistance
- Buy & sell AI services with bitcoin
- Join a network of ever-evolving AI tools
- Built-in Bitcoin wallet
- Nostr protocol integration

## Tech

- React Native + Expo via [Ignite](README-ignite.md)
- Open protocols
  - [Bitcoin](https://bitcoin.org/) & [Lightning](https://lightning.network/) via the [Breez SDK](https://sdk-doc-liquid.breez.technology/guide/about_breez_sdk_liquid.html)
  - [Nostr](https://github.com/nostr-protocol/nostr)
  - [Model Context Protocol](https://modelcontextprotocol.io/)
